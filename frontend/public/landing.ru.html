Частые вопросы

Что такое ставка финансирования (funding)?

Фандинг — периодический платёж между лонгами и шортами на бессрочных фьючерсах. Положительная ставка означает, что шорты платят лонгам — можно зарабатывать, просто удерживая нужную сторону позиции.

Как работает арбитраж фандинга?

Funding Finder находит пары, где ставка сильно отличается между двумя биржами (например, Binance и Bybit), и показывает спред. Открывая противоположные позиции на каждой бирже, вы забираете разницу, оставаясь рыночно-нейтральным — без ставки на направление.

Нужно ли пополнять счёт?

Нет. Сканы, поиск арбитража и 1 AI-идея в день бесплатны. Депозит нужен только если решите открыть позицию на бирже.

Безопасны ли мои API-ключи?

Ключи шифруются на сервере и используются в первую очередь для чтения. Автоисполнение запускается только с вашего явного разрешения.

Как оплатить?

В USDT через Crypto Pay и NOWPayments. Новым пользователям — 7 дней Pro без оплаты. Годовая оплата экономит 20%.

Есть ли API для разработчиков / фондов?

Да — версионированный публичный API (v1) отдаёт агрегированные данные фандинга 23 бирж, плюс white-label и встраиваемые виджеты. Смотрите раздел API / B2B.
